== Description ==

This plugin is a sidebar widget which can slide show photos in one of your picasaweb albums.

== Installation ==

1. Upload `picasaweb-photo-slide` directory to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Set up your username and album name.
